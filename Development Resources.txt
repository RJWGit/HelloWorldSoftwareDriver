Important tools/topics:
devcon- Can be used to install driver.

WDK-Windows driver kit is used to develop, test, and deploy Windows drivers.

KMDF- Kernel-Mode Driver Framework is a driver framework developed by Microsoft as a tool to aid driver developers create and maintain kernel mode device drivers for Windows


Other:
Target and host computers must be able to communicate with one another, check via ping command "ping 192.168.x.xx".
